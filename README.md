## ⚙️ 설치 및 실행
```
# 1. 클론
git clone https://github.com/username/chaji.git
cd chaji/chaji-client

# 2. 환경변수 설정
VITE_TMAP_API_KEY=본인_Tmap_API_Key
VITE_API_BASE_URL=http://localhost:8082

# 3. 패키지 설치
npm install

# 4. 실행
npm run dev

# 5. 브라우저 접속
http://localhost:5173
```
